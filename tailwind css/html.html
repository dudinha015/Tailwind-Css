Curso de HTML

Neste curso você aprenderá a criar a estrutura de páginas web utilizando HTML. Você verá como trabalhar com títulos, parágrafos, imagens, links, tabelas, listas, formulários e botões.

O HTML é a base de qualquer site e é o primeiro passo para quem deseja entrar no mundo do desenvolvimento web.

Duração 12h | Nível Iniciante

Este curso faz parte da trilha Front-End.
