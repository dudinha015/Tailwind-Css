Curso Front-End

Neste curso, você aprenderá como funcionam as principais tecnologias do desenvolvimento web: HTML, CSS e JavaScript.

HTML é utilizado para criar a estrutura da página, como títulos, textos, imagens e botões.

CSS é responsável pelo design da página, deixando o site bonito, organizado e responsivo.

JavaScript adiciona interatividade, como cliques, animações, menus, formulários e muito mais.

Informações do Curso

Duração: 15h

Nível: Iniciante

Certificado: Sim
