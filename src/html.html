Sobre o curso

O curso de HTML foi criado para quem deseja começar no mundo da programação e desenvolvimento web. Nele, você aprenderá como funciona a estrutura de um site, como criar títulos, parágrafos, imagens, links, botões, listas, formulários e muito mais.

O que você vai aprender?

  • Estrutura básica de uma página HTML
  • Uso de títulos, textos e links
  • Inserção de imagens e vídeos
  • Criação de formulários
  • Organização de conteúdo em páginas web
